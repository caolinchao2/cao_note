1、过渡属性： ​

  1. 属性名

    transition-property;

    不写默认是all，表示全部加入过渡的属性；

  2. 时间

    transition-duration;

    中间用逗号隔开,和过渡属性一一对应。

  • 语法 ​

css
特点 ： 不需要触发条件，浏览器在加载的时候就会执行动画，可以完成复杂的动画（可以插入多个关键帧） 语法 ：
第一步要制作动画的轨迹

*1.调用动画，设置要调用的动画的名称*/

animation-timing-function: linear;
3.动画的延迟时间*/
animation-delay: 0s; 
4.动画的所需的时间*/ 
animation-duration: 1s;
5.动画运动的次数 ：
number | infinite(无限次运动)*/
 animation-iteration-count: infinite; */
6.动画停留在最后一帧（注意：和infinite一块使用）*/
animation-fill-mode: forwards; 
7. 动画运动的方向 
normal| alternate(反方向也有动画)*
/ animation-direction:alternate ; 
复合写法 ： 
animation: name duration timing-function delay iteration-count direction fill- mode; 不要的属性可以去掉

四、filter ​

filter属性定义了元素的可视效果，比如：模糊与饱和度，通常给img标签设置

css 语法 ​

css
filter: none | blur() | brightness() | contrast() | drop-shadow() | grayscale() | hue-rotate() | invert() | opacity() | saturate() | sepia() | url();
filter: none | blur() | brightness() | contrast() | drop-shadow() | grayscale() | hue-rotate() | invert() | opacity() | saturate() | sepia() | url();
1

参数介绍 ​

  • none：没有效果

  • blur(px)：设置高斯模糊

  • brightness(%)：让图像更明亮或更暗淡

  • drop-shadow()：图片设置阴影效果

    css
    img {
    -webkit-filter: drop-shadow(8px 8px 10px red); /* Chrome, Safari, Opera */
    filter: drop-shadow(8px 8px 10px red);
}
    img {
    -webkit-filter: drop-shadow(8px 8px 10px red); /* Chrome, Safari, Opera */
    filter: drop-shadow(8px 8px 10px red);
}
    1
    2
    3
    4

  • contrast(%)：对比度，0的话图像全黑，1的话图像不变

  • grayscale(%)：灰度图像，1为完全转换灰度图像，0为无变化；

  • hue-rotate(deg)：改变图像的整体色调，0deg为图像无变化，超过360deg绕一圈；

  • invert(%)：反转输入图像，1为完全反转，默认为0

  • opacity(%)：转化图像的透明程度

  • saturate(%)：饱和度，0为完全不饱和，1无变化

  • sepia(%)：转换为深褐色，1为深褐色，0无变化

  • url()：接受一个xml文件，该文件设置svg滤镜，且包含一个具体的滤镜元素

    css
    filter: url(svg-url#element-id)
    filter: url(svg-url#element-id)
    1

  • initial：默认值

  • inherit：继承

复合函数 ​

使用多个滤镜，每个滤镜使用空格分隔

注意: 顺序是非常重要的 (例如使用 grayscale() 后再使用 sepia()将产生一个完整的灰度图片)。

css
img {
    -webkit-filter: contrast(200%) brightness(150%);  /* Chrome, Safari, Opera */
    filter: contrast(200%) brightness(150%);
}

    -webkit-filter: contrast(200%) brightness(150%);  /* Chrome, Safari, Opera */
    filter: contrast(200%) brightness(150%);

backdrop-filter ​

可以让你为一个元素后面区域添加图形效果（如模糊或颜色偏移）。 因为它适用于元素背后的所有元素，为了看到效果，必须使元素或其背景至少部分透明。

用法同filter；

媒体查询 ​

css
1.媒体查询 常见媒体尺寸设置：
/*超小型设备（手机，768px以下）*/
@media screen and (min-width:768px) { ... }
/* 小型设备（平板电脑，768px 以上） */
@media screen and (min-width:992px){ ... }/*
中型设备（台式电脑，992px 以上） */
@media screen and (min-width:1200px){ ... }/* 
大型设备（大台式电脑，1200px 以上） */ 
2. 响应式宽 （不确定的宽要用百分比） 
3. 响应式图片 
4. 响应式字体大小 
5. 设置viewport
